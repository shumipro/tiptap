## Front-end Development

### Install gulp

```
npm install -g gulp
```

### Run npm install
```
npm install
```

### Run gulp
Watch JS/CSS changes and build
```
gulp
```

Compress JS for production
```
gulp --production
```

### Access to localhost
http://localhost:8080/xxx.html  

* xxx.html is located in public folder *
